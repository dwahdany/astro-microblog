I made [a repo for tectonic that publishes it to pypi](https://github.com/dwahdany/tecto). I mentioned tectonic [in my last note](https://blog.wahdany.eu/2026/Apr/16/tectonic-typesetting/).

So, if you ever want to compile a tex file again, all you need to do is run `uvx tecto somefile.tex`!
